# Succulent Sphere — Next.js Storefront

Premium D2C ecommerce homepage built with Next.js, TypeScript, Tailwind CSS and Framer Motion.

Goals:
- Luxury minimal UI
- SEO & performance optimized (Next/Image, SSR where needed)
- Shopify Storefront API ready

Getting started:
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`

Environment:
- Provide `NEXT_PUBLIC_SHOPIFY_STOREFRONT_API` and `NEXT_PUBLIC_SHOPIFY_STORE_DOMAIN` when integrating with Shopify.

Folder structure:
See src/ — components, lib, styles, data.
